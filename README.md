# Galería de Superhéroes

## Funcionalidades
- Renderizado dinámico de cards desde un arreglo
- Formulario para agregar personajes
- Eliminación de personajes
- Filtro por nombre

## Tecnologías
HTML5, CSS3, Bootstrap 5, JavaScript (DOM).

## Conceptos aplicados
- Objetos y arreglos (arreglo personajes como única fuente de verdad)
- Funciones declarativas y funciones flecha
- Desestructuración de objetos
- Template literals
- Manipulación del DOM (getElementById, innerHTML, addEventListener, dataset)
